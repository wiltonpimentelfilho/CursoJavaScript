Funções Matemáticas

Existem diversos tipos de funções matemáticas, as mais comuns são:

1. Função Math.round() - Arredonda as casa decimais para a média.
2. Função Math.ceil() - Arredonda para cima.
3. Função Math.floor() - Arredonda para baixo.
4. Função .toFixed(x) - Organiza em "x" casas decimais.
Exemplo:

function calcDiaria( ){
    return Math.round(5000/30);
}
var diaria = calcDiaria();
alert(diaria); <-- "Aqui estamos chamando a execução dessa função."
