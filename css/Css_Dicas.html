Página de Dicas sobre o uso do Cascading Style Sheets


Seletores CSS

Utilizando os seletores css fica mais fácil e prático formatar os arquivos html. Ex:

h1 {
color: black;
border: 1px solid blue;
text-align: center;
}

Classes e IDs = class e ID

Para escrevermos uma classe adicionamos o "." seguido do nome da classe, seguida de "{ }", contendo os atributos que desejarmos utilizar.
Para criarmos um ID, utilizamos a "#" precedido do nome do ID, seguido de "{ }", contendo os atributos que desejarmos utilizar.
Obs: Você pode utilizar as class, os ids, os styles, junto as tags diretamente no arquivo "html". A isso chamamos de "Inline CSS".
